(1)請使用靜態遞迴方法算出費氏數列。
```
public class ex01 {
    public static void main(String[] args) {
        for (int counter = 0; counter <= 10; counter++){
            System.out.printf("Fi: %d\n", counter, fibonacci(counter));
        }
    }
 
    public static long fi(long number) {
        if ((number == 0) || (number == 1))
            return number;
        else
            return fi(number - 1) + fi(number - 2);
    }
}
```
(2)請使用 iterative方法算出費氏數列。
```
public class ex02 {
    public static long Fi(int n)
    {
        long v1 = 0;
        long v2 = 1;
        long result = n;
        for (int i = 2; i <= n; ++i)
        {
            result = v2 + v1;
            v1 = v2;
            v2 = result;
        }
        return result;
    }
}
```
